Rack Server Pages
=================

Rack server pages is a rack middleware that serve a view templates directly with tilt.
There are no controllers and no models, just only views like a asp, jsp and php!

## Features
- render with tilt
- mime-type
- sample.php :)
- path regex
- rack request
- session
- params
- scope
- partial template
- layout template
